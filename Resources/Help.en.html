ReportBuddy - User Guide

Initial Setup

  1. Select calendars to monitor from the "Calendars" dropdown in the sidebar
  2. Add keywords to filter specific events
  3. Configure at least one email template from Preferences (⌘,)

Event Filters

  • Keywords:
    • Events are filtered based on entered keywords
    • Double-click a keyword to edit it
    • Click the ✕ symbol to remove it
  • All-day Events: Enable this option to show only all-day events
  • Date Range:
    • Default: current month
    • Custom: select specific start and end dates

Email Templates

Access template settings from ReportBuddy > Preferences or using ⌘,
  • You can create multiple templates for different types of reports
  • Each template can have:
    • A default recipient
    • A customized subject with variable support
    • A header and footer message
    • A specific language (Italiano or English)
  • Available variables in subject:
    • $month - Current month
    • $year - Current year

Keyboard Shortcuts

  • ⌘, - Open Preferences
  • ⌘R - Refresh Events
  • ⌘E - Export via Email
  • ⌘Q - Quit Application
  • ⌘H - Hide ReportBuddy
  • ⌘⌥H - Hide Other Applications
