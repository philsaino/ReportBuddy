## System Requirements
- macOS 13.0 or later
- Calendar access
- Configured email account

## Installation
1. Download the latest release
2. Open the .dmg file
3. Drag ReportBuddy to Applications folder
4. Grant calendar access when prompted

## Privacy

- No external data transmission
- Local configuration storage
